# Restful Booker QA Portfolio Project

## Project Overview

This portfolio project demonstrates practical software testing skills through manual web testing, API testing, defect reporting, and SQL data validation.

The project will include test planning, test-case design, exploratory testing, bug reporting, API testing with Postman, SQL validation, and a final test summary.

## Application Under Test

Restful Booker Platform is a demonstration bed-and-breakfast booking application designed for software testing practice.

## Planned Testing Areas

- Functional web testing
- Exploratory testing
- Positive and negative testing
- Form validation
- Booking-flow testing
- API testing with Postman
- SQL data validation
- Defect reporting
- Test-summary reporting
